# Monetary Policy Shocks in the Euro Area: A FAVAR Approach

This repository contains the code and data used to estimate a 
Factor-Augmented Vector Autoregression (FAVAR) model for the euro area.

The project follows the methodology of Bernanke, Boivin and Eliasz (2005)
and compares standard VAR and FAVAR impulse responses.

## Data
Monthly euro area macroeconomic and financial variables (2000â€“2025).

## Methodology
- Principal Component Analysis for factor extraction
- Bernanke et al. (2005) identification strategy
- Cholesky identification of monetary policy shocks
- Robustness checks on factor dimension and shadow rate

## Output
- Impulse response functions
- Cumulative responses
- Variance decomposition
